# Open Text Collections #3
## Jarawara texts

This collection features 20 texts from the Jarawara language, spoken in Brazil. 

The texts were narrated by two speakers called Yowao and Siko. The stories were compiled, translated, and glossed by Alan Vogel. 

The files in this repository contain the texts in the CLDF format. A human readable version of the book is available at https://zenodo.org/records/17158276
